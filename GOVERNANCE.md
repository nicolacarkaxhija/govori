# Governance

Govori is maintained by [Nicola Carkaxhija](https://github.com/nicolacarkaxhija).
This document states the guarantees that keep the project — and especially the
community's contributed content — safe from depending on any single person.

## Continuity guarantees

1. **The content is never hostage.** All learning content is licensed
   CC BY-SA 4.0 and continuously exported to a public repository. The code is
   AGPL-3.0. Anyone can rebuild the platform from public artifacts at any time.
2. **Succession promise.** If the maintainer is unreachable for **six
   consecutive months** despite documented contact attempts (GitHub issues,
   e-mail), stewardship of this repository transfers to the project's
   co-administrators, or — once a partnership with the Interslavic language
   community is established — to successors nominated by that community.
3. **Co-administration.** As soon as a contributor has earned sustained trust,
   they will be granted repository co-admin rights so that reviews, merges, and
   releases never depend on one calendar.
4. **Credentials escrow.** Domain, server, and store credentials are held in a
   password manager with an emergency-access contact, so operational handover
   does not require archaeology.

## Roles

| Role          | Scope                                                         |
| ------------- | ------------------------------------------------------------- |
| Maintainer    | Direction, releases, infrastructure, this document            |
| Co-admin      | Merge, review, release; emergency stewardship                 |
| Content roles | Contributor / Reviewer / Admin — defined in-app, see ADR 0008 |

## Changing this document

Changes to the continuity guarantees are announced publicly and never take
effect retroactively.
